## 添加图像处理

对于想要了解图像处理，首先应当对图像的形成有一定的认知和了解。主要有：
> * 图像成像原理
> * CMOS 和 CCD 的区别
> * RGB 以及对应相关的转换
> * 图像噪声产生的原因  

对于图像处理需要更多在数学上的理解，而不是只是应用层。主要要理解:
> * 像素的操作、通道分离和合并
> * 直方图以及直方图
> * 膨胀、腐蚀、开运算、闭运算
> * 基本滤波、Sobel 算子、拉普拉斯算子，Canny 算子
> * 轮廓检测
